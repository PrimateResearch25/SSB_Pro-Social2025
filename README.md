code - All code used in the analyses

23_24_25_raw_scans.csv - All scan data collected from 2023, 2024, and 2025. Used to calculate observability and encounter frequency

control_variables_23_24_25.csv - Control variables for each individual male macaque, including observability, rank (David's score), and age.

ior_23-25_organised_collapsed.csv - Inter-observ reliability dataset, organised ready for analyses. This version collapses the behavioural events using an adjacent mirror merge to avoid issues with timestamps caused by recording device differences. Used in the IOR analysis.

raw_aggression_mount_23_24_25.csv - All aggression and mounting observations across 2023, 2024, and 2025.

raw_groom_mount_23_24_25.csv - All grooming and mounting observations across 2023, 2024, and 2025.

raw_male_aggression_mount_23_24_25.csv - All male-male aggression and mounting observations across 2023, 2024, and 2025.

raw_male_coalition_mount_23_24_25.csv - All male-male coalition and mounting observations across 2023, 2024, and 2025.

raw_male_groom_mount_23_24_25.csv - All male-male grooming and mounting observations across 2023, 2024, and 2025.

winner_loser_interactions_23_24_25.csv - All male-male aggression and submission observations across 2023, 2024, and 2025. Used to calculate David's score.
